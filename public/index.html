Set Up Roblox Cookie

The Roblox economy API requires authentication to look up item owners. Your cookie stays on your machine and is only sent to Roblox's official API.

  1. Open Chrome/Firefox and go to roblox.com
  2. Log in to your account
  3. Open DevTools (F12) → Application tab → Cookies
  4. Copy the value of .ROBLOSECURITY
  5. Create server/data/config.json with:
{
  ".ROBLOSECURITY": "your_cookie_here"
}

Restart the server after saving. The cookie is in .gitignore.
